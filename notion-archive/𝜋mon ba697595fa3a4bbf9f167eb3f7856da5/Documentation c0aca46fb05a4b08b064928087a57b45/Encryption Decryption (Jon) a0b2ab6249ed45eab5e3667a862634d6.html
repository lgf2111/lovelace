Encryption/Decryption (Jon)

Encryption (encryptAES)

  • Accepts plainText as a dynamic data type (can be any type)
  • Randomly generates a 256-bit secret key and 128-bit IV each time the function is called.
  • This ensures the uniqueness of the secret key & IV whenever a new backup is made.
  • Secret key & IV are stored in base64 encoded String in Flutter_secure_storage after being created.
  • Flutter_secure_storage uses AES encryption for Android & RSA for iOS (KeyStore)
  • Returns the ciphertext in base64 for local storage

Decryption (decryptAES)

  • Accepts cipherText as a base64 String because the data passed into this function is read from the local JSON file, lovelace.json and its data type is base64 String.
  • Gets the key and IV from Flutter_secure_storage as a base64 String.
  • Decryption only works if a key of data type Key and IV of data type IV are used on a Decrypted cipherText. Otherwise, decryption will fail.
  • Converts the base64 Key from String to Key.
  • Converts the base64 IV from String to IV.
  • Decodes the base64 String argument, cipherText, and converts it from base64 String to Encrypted.
  • Decrypts data with the key and IV retrieved from secure storage and returns the plaintext as String.
